# Product Context

## Why This Project Exists

PhotoViewer, Windows kullanıcıları için yerleşik fotoğraf görüntüleyicilerin yetersiz kaldığı noktalarda devreye girer. Kullanıcıların fotoğraflarının metadata bilgilerini görmesi, temel düzenlemeler yapması ve albümlerini kolayca gezinmesi hedeflenir.

## Problems It Solves

1. **EXIF Bilgisi Eksikliği**: Windows fotoğraf görüntüleyicileri genellikle detaylı EXIF verisi göstermez
2. **Düzenleme İhtiyacı**: Basit döndürme veya kırpma için Photoshop gibi ağır uygulamalara gerek olmamalı
3. **Albüm Gezinme Zorluğu**: Klasör içinde tek tek fotoğraf açma zahmetli
4. **Tekil Çalışma**: Aynı fotoğrafı birden fazla uygulama ile açma karışıklığı

## How It Should Work

- Kullanıcı bir fotoğraf dosyası seçer veya sürükler
- Fotoğraf merkezi olarak gösterilir
- EXIF bilgileri sağ panelde görüntülenir
- Toolbar ile döndürme, kırpma, silme, kaydetme yapılır
- Fare tekerleği ve çift tıklama ile zoom yapılır
- Sol/sağ tıklama alanları ile albümde gezinilir

## User Experience Goals

- Hızlı ve akıcı performans
- Minimalist ve modern arayüz
- Sezgisel navigasyon
- Bilgilendirici ancak kalabalık olmayan metadata paneli
